✨ Features

⚡ Ghostty

GPU-accelerated terminal built with Zig. Native macOS feel with Metal rendering.

🟣 Zellij

Modern terminal multiplexer with Alt-based keybindings, floating panes, and WASM plugins.

⭐ Starship

Cross-shell prompt showing git status, language versions, and command timing.

🎨 Catppuccin

Beautiful purple-tinted theme that's easy on the eyes during long coding sessions.

🚀 Quick Start

git clone https://github.com/irfancode/dotfiles.git ~/dotfiles
cd ~/dotfiles && bash scripts/install.sh
exec zsh
